SEED Labs network security exercises covering encryption, traffic analysis,

This repository contains a collection of network security labs completed
using the SEED Labs framework as part of academic coursework.

## Topics Covered

- Encryption and secure communication
- Packet inspection and traffic analysis
- Man-in-the-middle (MITM) attack mechanics
- ARP cache poisoning and ICMP behavior

## Environment & Ethics

All experiments were conducted strictly in isolated virtual environments
using Docker containers and private IP ranges.  
No real networks were targeted, and no unauthorized systems were accessed.
