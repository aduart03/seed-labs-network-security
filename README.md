The labs explore foundational security concepts including:

- Encryption and secure communication
- Packet inspection and traffic analysis
- Man-in-the-middle (MITM) attack mechanics
- ARP cache poisoning and ICMP behavior

All experiments were conducted in isolated virtual environments using
Docker containers and private IP ranges for educational purposes only.
